# Lakeview Widget Negatives (詳細カタログ)

ローカル保守用。Databricks workspace にはデプロイされない (sync-to-genie.sh --init-all は SKILL.md のみアップロード)。新規 pitfall 発見時は SKILL.md を先に更新し、本ファイルも同期する。

## bar

| 非対応機能         | 詳細                                         | 代替策                                | 確認した実例                    |
| ------------------ | -------------------------------------------- | ------------------------------------- | ------------------------------- |
| 軸のカスタムソート | `axis.sort` で任意順序指定不可               | SQL `ORDER BY` で並び SQL 側固定      | 統括会議 v10                    |
| データラベル表示   | `mark.showLabels` / `bar.label` が無視される | tooltip で代替、または title でサマリ | 統括会議 v11 構成比 stacked bar |
| Marimekko          | 幅と高さ両方をカテゴリで変える表現           | bar + 別カラム                        | -                               |
| Y軸目盛りカスタム  | `axis.values` / `axis.tickCount` が効かない  | データを SQL で絞る                   | 統括会議 v12                    |
| 連結線 (Mekko)     | 棒同士を線で繋ぐ表現 不可                    | line widget に切り替え                | 統括会議 v12                    |

## line

| 非対応機能        | 詳細                                  | 代替策                                   | 確認した実例 |
| ----------------- | ------------------------------------- | ---------------------------------------- | ------------ |
| セグメント連結    | NULL 行で線途切れ (gap bridging なし) | SQL 側で `COALESCE(col, 0)` で NULL 埋め | -            |
| 多色凡例の順序    | 凡例カスタム順序不可                  | SQL でラベル接頭辞 (`'01_xxx'`)          | -            |
| 補助線 (平均線等) | 横ライン追加 不可                     | 別 layer の line を重ねる (要設計)       | -            |

## table

| 非対応機能             | 詳細                       | 代替策                                | 確認した実例                         |
| ---------------------- | -------------------------- | ------------------------------------- | ------------------------------------ |
| encoding `type` 省略   | 列が空欄になる (推論失敗)  | `"type": "quantitative"` 等を必ず明示 | 統括会議 (type: "number" 明示で空欄) |
| 行クリックリンク       | カスタム link 不可         | 別ダッシュボードへの link tile で代替 | -                                    |
| ソート方向のデフォルト | 列単位ソート方向の固定不可 | SQL `ORDER BY` で初期表示順を固定     | -                                    |

## pivot

| 非対応機能         | 詳細                            | 代替策                            | 確認した実例                     |
| ------------------ | ------------------------------- | --------------------------------- | -------------------------------- |
| セル内条件付き書式 | セル値による色付け 不可         | 色付け不要なメトリクスに絞る      | -                                |
| 小計行             | 自動小計 不可                   | SQL `UNION ALL` で小計行展開      | -                                |
| 行・列の動的入替   | UI で自由入替 不可 (定義時固定) | rows / columns を要件確定後に設計 | 統括会議 v7 船坂シート風で要設計 |

## counter

| 非対応機能        | 詳細                                   | 代替策                         | 確認した実例 |
| ----------------- | -------------------------------------- | ------------------------------ | ------------ |
| 複数値並列        | 1 counter で複数 KPI 不可              | counter を複数配置             | -            |
| 副表示 (前年比等) | 比較値の小さく表示 不可                | counter + bar/line             | -            |
| 単位記号          | "%" や "円" などの suffix カスタム不可 | SQL 側で `CONCAT(value, '円')` | -            |

## pie

| 非対応機能     | 詳細                 | 代替策                         | 確認した実例 |
| -------------- | -------------------- | ------------------------------ | ------------ |
| 10+ カテゴリ   | 凡例で潰れて読めない | SQL で TOP N + "その他" に集約 | -            |
| ドーナツ       | 内側ラベル不可       | bar に変更検討                 | -            |
| カテゴリ並び順 | 凡例順カスタム不可   | SQL でラベル接頭辞 で順序制御  | -            |

## メンテナンス

新しい pitfall を踏んだら:

1. SKILL.md (workspace へデプロイされる) を先に更新
2. 本ファイルの該当 widget 表に行追加 (実例列に該当タスク・バージョンを記録)
3. lakeview-handoff.md.tmpl の "既知制約" セクションへの追記が必要なら検討
